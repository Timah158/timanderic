# What is this project?

The goal of this project is for the app's creators, Tim and Eric, to both be able to improve thier skills and showcase our latest accomplisments.
By utilizing thi application we hope that we can not only show what we have accomplised, but will also be driven to persue greater ideas.

### How does it work?

1. We deployed the app in AWS using a LightSail instance.
2. We configured the LightSail instance to use SSL and NodeJS.
3. We configured React on the server and deployed this GitHub project on it.

### What does it do?

1. The application show cases our skills and experience to potential employers.
2. This will make it easy to contact us. Looking for a resume? **Bam!** It's right there.
3. It give us, the creators a project to work on and a means for organizing projects.
4. We will have more drive to work on projects as they will be publically documented.
